This is a 3rd person stealth game where the goal is to infiltrate the main building. Starting in the parking lot, you must find a key to unlock the gate
and sneak passed the cameras and guards. Inside the courtyard, there are 3 patrolling guards and another key to find to get passed the last gate. 

Interact is left click, can be hard to aim in its current state as it is pretty precise with no crosshair.
